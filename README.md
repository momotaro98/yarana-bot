# yarana-bot  やらなボット

Line Bot Application

## Feature

* やると褒めます
* やらないと怒ります

This bot
* praises you if you do your task
* gives you a scolding if you don't

## Development

###  Build app

```
$ go get github.com/momotaro98/yarana-bot
$ cd $GOPATH/src/github.com/momotaro98/yarana-bot
$ make
```

### Run unit test

```
$ make test
```
